# 应届全职岗位筛选记录

候选人定位：

```text
26届本科
山东第一医科大学
医学信息工程
目标城市：苏州优先，其次合肥/苏州周边
目标方向：Java后端、工业软件、MES、IoT平台、制造业数字化、设备数据平台
接受程度：偶尔现场沟通可以，不接受长期驻厂
薪资底线：至少 5-6K，优先 7K+
```

## 筛选原则

优先投：

```text
应届/经验不限/1年以内
Java/Spring Boot/Vue/MySQL/Redis
工业软件/MES/ERP/IoT/设备管理/制造业数字化
苏州、昆山、吴江、工业园区、相城、合肥
薪资 6K 以上
```

可以冲：

```text
写 1-3 年经验，但技术栈和项目高度匹配
制造业软件公司，要求可适当放宽
岗位描述里有 Java、Spring Boot、Vue、接口、系统开发、设备数据
```

谨慎投：

```text
大量出差/驻厂
纯实施顾问
低代码为主
不是软件开发主岗
薪资明显低于生活成本
```

不建议：

```text
3-5年或5年以上硬要求
纯测试、纯运维、销售、售前
嵌入式底层、C/C++硬件驱动为主
PLC/电气/机械结构为主
```
